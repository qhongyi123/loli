# 小贴士更新说明

## 数据文件

| 文件 | 作用 |
|------|------|
| `data/tips.js` | **实际加载的数据文件**，修改后直接生效 |
| `data/tips.json` | 纯 JSON 格式参考（不会被程序读取，仅对照格式用） |
| `data/tips.example.json` | 示例格式 |

## 格式要求

`data/tips.js` 第一行固定为 `window.__LOLI_TIPS__ = `，后面跟 JSON 数组，末尾分号。

每项包含两个字段：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id   | 数字 | 是   | 唯一编号 |
| text | 字符串 | 是   | 小贴士文字内容 |

## 如何添加新小贴士

1. 打开 `data/tips.js`
2. 在数组的 `];` 之前追加新条目，用逗号分隔
3. 确保 `id` 不与已有条目重复
4. 保存文件

### 添加示例

```js
window.__LOLI_TIPS__ = [
  { "id": 1, "text": "测试对话1" },
  { "id": 2, "text": "测试对话2" },
  { "id": 3, "text": "测试对话3" },
  { "id": 4, "text": "我新加的小贴士" },
  { "id": 5, "text": "又一条新内容" }
];
```

### 删除

从数组中移除对应行即可，无需修改其他条目的 id。

### 修改

直接修改对应条目的 `text` 字段即可，无需改 id。
